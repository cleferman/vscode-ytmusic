# vscode-ytmusic

Connecting VS Code with the [YouTube Music Desktop Player](https://ytmdesktop.app/)

## Features

* See what song is currently playing
* Play / Pause
* Skip / Rewind
* Cycle Repeat Mode

## Requirements

* [YTMDesktop App](https://ytmdesktop.app/)

## Building

Run `vsce package` to build the extension.
