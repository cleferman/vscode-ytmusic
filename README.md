# vscode-ytmdeskptop

Connecting VS Code with the [YTMDesktop Player](https://ytmdesktop.app/)

## Features

* See what song is currently playing
* Play / Pause
* Skip / Rewind
* Cycle Repeat Mode

## Requirements

* [YTMDesktop Player](https://ytmdesktop.app/)

## Building

Run `vsce package` to build the extension.
